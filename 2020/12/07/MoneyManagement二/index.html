2020年理财直播课

2020.10.7 直播课—挑选基金，知道这几个诀窍

  1. 宽基指数和行业指数？
    • 宽基：覆盖面广、个股权重低；
    • 沪深300，中证500；
    • 证监会划分的行业：11个行业
  2. 全球三大长牛行业？波动性：消费 < 医药 < 科技
    • 医药行业：美国生科、A股300医药（全民医保）；
      • 医疗ETF、生物医药ETF、创新药ETF
    • 科技行业：纳斯达克综合指数、A股科技ETF（要向美国靠齐）；
      • 科技龙头ETF、新能车ETF（未来10年）、芯片ETF（国家战略倾向）
    • 消费行业：标普美国品质消费、A股300消费（消费大国、人民群众美好的向往）
      • 消费50ETF、消费ETF、酒ETF
  3. 牛市来了，券商ETF属于高弹性的周期之王
    • 券商ETF：每次牛市来的时候都会涨；
    • 吸引外资：外资券商企业可以进入中国A股进行控股，券商的合并潮会拉动券商ETF的发展；
  4. 其他值得关注的行业指数
    • 环保ETF：十四五规划，在未来的两三年是一个短牛；
    • 军工ETF：军工预算逐步增加，在A股上市的军工企业不是最优秀的企业（国家机密的行业不会上市）；
  5. 美股VS 中国A股
    • 美股：长牛
    • A股：熊牛穿插，熊长牛短；峰回路转，最大回撤72%；
    • 疯牛（有两个高峰顶），熊市（就会一直下跌）
  6. 指数基金的投资方式
    1. 长期定投：3-5年的定投阶段，2-3月的兑现阶段；
      • 下一个牛市是2021年，因此现在定投得抓紧时间；
    2. 中期波段：3-6个月进行操作
      • 付费社群里面的操作；
    3. 恐惧贪婪指标（量化模型）：抄底逃顶神器—理性分析，避免主观判断
      • 20分以下——行情处于恐惧；
      • 80分以上——行情处于贪婪；
      • 每天公布两次：上午的公布（前天），下午2:30公布（当天）；

  1. 市场的第四个季度？
  2. 节前和节后？

2020.11.22 直播课——震荡行情

  1. 近期大盘走势
    • 弱势震荡，无主跌风险；每天成交量在8000亿左右（好的行情在10000亿左右）；
    • 核心指数呈现沪强深弱；
  2. 东方财富全A：所有的A股指数行情；
    • 目前都在区域里面震荡：周一上涨后进行调整；
    • 沪深300：目前只有沪深突破震荡区间；
  3. 两种行情如何做？
    • 震荡行情：高抛低吸；
    • 突破行情：等到压力位跑路；
  4. 创业指数：目前行情比较低，调整的行情还没有结束，因此不要去抄底；
    • 创业指数构成：医药行业
    • 如何进行购买：分档进行购买，每降3个点买一档；
    • 白酒和医药千万不要去抄底
  5. 深圳指数
    • 如何确定是都在某个顶部站稳，可以突破高点：至少在两个交易日在高位交易日；
  6. 这个月的指数突破概率不大，其他指数可能会在12月初会突破；
  7. 证券化率
    • 现在的A股并不便宜：因此这波行情过后，A股可能会下跌很长一段时间；当这轮行情结束后资金会在2月份左右会出来；
    • 医药、科技：震荡的行情；
    • 买房：房子是刚需的话，今年确实应该买了；房子已经压了很低了；
  8. 市场震荡，资金方向在哪里配置？
    • 顺周期板块：经济复苏后建筑板块上涨；
    • 半导体、新能源车：潜力较高的板块；
    • 医药：医药认识未来市场长期主线；但是现在不要抄底；
  9. 如何抓住买入时机？

2020.12.7 在线直播答疑专场

基金投资问题

  1. 关于大盘：指数几次冲击上沿都没有突破，如何看待后面的市场？
    • 沪深300、上证50：创建了新高；
    • 从7月份到现在的12月：7月创新高后，大部分指数都在区间内震荡；
  2. 疫情：对市场进行了一次淘汰，留下更多资金雄厚的店家；
  3. 央行每一天的操作？
    • 放几天水，然后又收几天水；—— 震荡操作；
  4. 北向资金：过去四周都是净流入的；
  5. 关于ETF
    1. 券商ETF
      • 目前是突破了下跌的趋势的；
      • 券商疯长时：代表了一轮行情即将结束；
    2. 芯片ETF：基地很牢靠，不太适合做高抛低吸的，会把资金丢掉；
    3. 新能车、光伏模块：对标美股的新能车，不确定性比较高，但是基地也在构建；
    4. 创新药ETF、生物医药ETF、医疗ETF：创新药的技术层次更高

2020.12.18 解读12.11政治局会议

会议内容

  1. 每年11月12月会开一个政治会议，为来年的经济做一个布局；
  2. 2021年整体定位很高？
    • “十四五”开局之年；
    • 建党100周年，因此明年的电影会有很多关于党的电影；
    • 全面建成小康社会的宣布、评估与总结之年；
    • 2022年是二十大的人事布局之年（十九大结束）；
  3. 疫情和外部环境依然又不确定性？
    • 经济和货币政策不会做大幅度的变化；
      • 今年GDP是正增长；
    • 拜登上台：对澳洲实行限制政策；
  4. 需求侧改革？
    • 供给侧改革：通过环保的方式，淘汰落后产能；
    • 需求侧改革：防止疫情结束后，产业会大量倒闭可能；
    • 调整需求侧，构建双循环链，国内国外的供应链改革；
  5. 增强产业链自主可控，包括半导体；

会议首提

  1. 注重需求侧改革？
    • 核心：扩大内需，抓手是畅通内循环、出台促销费举措、推动配套制度改革；
    • 目的：“衣食住行”，改革方向—引导富人们在国内消费（富人出不去了）。
      • 衣：快时尚，淘宝为购物首选；
      • 食：高端食材、高品质食材（疫情影响进口）；
      • 行：新能源汽车，汽油是我国一个短板；（上海引入特斯拉，上汽研发）
      • 学区房：核心区域、核心城市学区房涨价；
        • 没有提“房住不炒”：豪宅可以炒，和普通人没有关系；
        • 改善型住宅：逐步成为房地产的主流；
        • 刚需房子：不允许炒房，大量建立公租房；
      • 国际学校、民办教育：国内的学校有意愿和国外学校进行合作进行办学；
        • 举例：珠三角地区，现在房地产；
    • 举例：社区团购—扼杀了小商小贩的失业；
  2. 首提“强化反垄断和防止资本无序扩张”，互联网巨头首当其冲？
    • 消费举措：减少年轻人的借贷消费，摁下蚂蚁金服；
    • 消费能力：信贷能力；
    • 举例：90年代日本人，通过借贷后进行消费；在未来的3年后消费会大幅度下降；
    • 人会穷，因为人没有自律，不回去储蓄；
  3. 富人在国内会消费的地方？
    • 豪宅、美女、豪车、教育、医疗；
  4. 首提“抓好各种存量风险化解和增量风险防范”？
    • 稳杠杆、防风险是明年重点，2020年下半年；
  5. 会议重要精神？
    • 我国整体经济形势不错；
    • 2021年政策稳中渐退；
    • 重视需求侧改革，扩大内需；
    • 2021年加大监管力度：该查的查，该防的防。

恐贪指数和基底

  1. 恐贪指数是什么？
    • 3000支股票的一个基本走势；
    • 恐惧的时候买，贪婪的时候卖，平时就可以不动。
  2. 基底是什么？
    • 基底：长时间在一定范围内进行上下波动；（K线上下波动）
    • 作用：用于大量吸收资金；
    • 收盘价：判断K线的最主要条件；

  3. ETF和指数的区别：总体和局部的关系；

    • 现在跟踪的是ETF，并不是某一只指数；
    • 跌破基底止损：出现过一轮行情的时候，才需要止损；
    • 突破基底止盈：突破收盘价或开盘价的最高上沿；

  4. 港股打新？开港股账号进行好公司投资；

    • 不是每个新股都能打新；
  5. 茅台：需求侧改革的典型案例，用来收富商的税。
    • 消费的基金可以买入了；
