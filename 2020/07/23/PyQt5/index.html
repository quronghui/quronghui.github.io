第四章 PyQt5 基本窗口控件

1. 窗口类型介绍

  1. 窗口类型的介绍：
    • QMainWindow：GUI的主窗口，包含菜单栏（menuBar）、工具栏(addToolBar)、状态栏(statusBar)、标题栏；
    • QDialog：对话框窗口基类，用来执行短期任务，或者与用户进行互动。没有菜单栏……
    • QWidget：如果窗口不确定时都作为顶层窗口或者嵌入窗口，便可以使用QWidget；所有窗口和控件都间接或直接继承该类；
  2. QMainWindow：GUI的主窗口
    • showMessage（）显示消息，并延时
    • screenGeometry（）获得屏幕的大小，并将窗口放在中间
    • quit（）关闭窗口；
      • text()函数：获取显示的名称；
      • print()输出可以在Terminal串口里面查看信息
  3. QWidget
    • geometry()：获取窗口的坐标系
    • sys.exit():进入程序的主循环
    • QIcon()：在主窗口的标题栏设置一个图标
    • setToolTip()：当鼠标在窗口里面是，提示一行文字；

2. QLable标签的使用

  1. QLable 标签的介绍

    • QLable对象作为一个占位符可以显示不可编辑的文本或图片，继承自QFrame;

  2. QLable类的一些方法

    • setAlignment()：设置文本对齐方式；
    • setPixmap()：设置标签为一个pixmap图片；

  3. QLable标签创建快捷键的方式

3. 文本框类型控件

  1. QLineEdit 类
    • 单行文本控件，输入单行字符串；
    • 可是设置掩码规则，和相应的文本验证器，用于对输入的文本内容进行验证；
    • 掩码：掩码字符和分隔符字符串组成，后面跟一个分号和空白组赴；
  2. QlineEdit类的一些方法
    • setPlaceholderText()：设置文本框浮显字符，起到提示作用，当输入后消失；
    • setEchoMode()：设置文本框显示格式，允许输入的文本显示格式有具体的格式要求；
      • QLineEdit.Normal：显示所输入的字符，默认选项
    • setInputMask()：设置掩码的格式；
    • setValidator()：设置文本的验证器（验证规则），将限制任意可能输入的文本；
  3. QTextEdit 类
    • 文本框：多行文本输入，当内容超过文本框时，显示滚动条；
  4. QTextEdit 类的方法
    • setPlainText()、toPlainText()：设置多行文本框的文本内容、返回多行文本框的文本内容；
    • setHtml()、toHtml()：设置文本框内容为HTML网页格式、返回多行文档内容；

4. 按钮类控件

  1. QAbstractButton：按钮类空间的基类；包含的按钮类包括？
    • QPushButton、QToolButton、QRadioButton、QCheckButton；
  2. QPushButton 类的一些方法；
    • 每次点击按钮时都会发射；
    • setCheckable()：设置按钮是否已被选中,保持点击和释放的状态
    • setIcon()：方法接收一个QPixmap对象的图像图件作为参数输入
    • setEnabled() ：是否可以使用，设置其不触发，显示就为灰色
    • print() 按钮的状态提示；
  3. QRadioButton 类：
    • 状态改变时才触发—有利于监控
    • setChecked（）：设置复选框的状态，设置为True是表示选中
  4. QCheckBox类：
    • 提供了一组带文本标签的复选框，用户可以选择多个选项；setIcon：设置图片；setText：设置文本
    • QGroupBox将多个复选框添加到一个水平布局管理器中，并添加到groupBox组里面
    • setTristate()函数提供三种状态的选择；0-选中，Qt.PartiallyChecked 属于半选中状态；
    • isChecked() 查询是否被选中；checkState() 选中的状态（0, 1,2）
    • str() 函数将对象转化为适于人阅读的形式

5. 下拉列表框

  1. QComboBox：下拉列表框，是一个集按钮和下拉选项一体的控件；
  2. QComboxBox 类的方法
    • addItem()添加单个选项； addItems()添加多个选项；
    • count（）返回下拉选项集合中的数目；
    • currentText 返回选中选项的文本；

6. 计数器

  1. QSpinBox：计数器的控件，允许输入整数或者浮点数，通过上下箭头增加/减少当前的值；
    • （1）QSpinBox 用于设置整数值，默认范围是0-99；
    • setMinimum() ,setMaximum() 设置整数计数器的边界值；
  2. QDoubleSpinBox 用于设置浮点数，默认两位精度；
    • setDecimals() 设置浮点计数器精度的位数

7. 滑动条

  1. QSlider滑动条：提供了一个水平或垂直的滑动条，这种方式可以呈现一种更好地效果；
  2. 具体的用法参考代码就行，所有的函数都包含了；

8. 对话框类控件

  1. QDialog
    • WindowModality窗口的模态设置（三种模态），Qt.ApplicationModal不与其他窗口交互；只有关闭了所有的子窗口后才能关闭窗口
    • 当用户按下Esc按键时，对话框窗口将会调用QDialog.reject()方法，用于关闭对话框
    • 主窗口QMainWindow，通过一个自定义函数包含了子窗口QDialog;
  2. QMessageBox：通用的弹出式对话框，用于显示消息
    • 功能都是一样的：起到一个提示作用；
    • infomation信息框 ：有标准的形式；
    • 实现了五种对话框的代码；
    • 问题：没有实现布局？解决layout = QVBoxLayout(self) 加入self变量，对其进行布局；
  3. QInputDialog：标准对话框，由一个文本框和两个按钮（OK，Cancel）组成；
    • 当用户按下OK或Enter后，父窗口可以收集QInputDiadlog输入的文本信息；
    • getInt()，getDouble()：从控件中获得标准整数、浮点数的输入；
    • getText()：从控件中获得标准字符串的输入；
    • getItem()：从控件中获得列表里的选项输入；
  4. QFontDialog：控件是一个常用的字体选择对话框，可以让用户选择所显示文本的字号大小、样式、格式
    • getFont()：从字体选择对话框中选择文本的显示字号、字体、样式、格式；
    • setFont()：设置需要改变字体的具体格式；
  5. QFileDialog：控件用于打开和保存文件的标准对话框；
    • 文件的打开和保存；
    • getOpenFileName()调用文件对话框显示图像；
      • self：显示父组件；第二个参数：对话框的标题；第三个参数：对话框显示默认打开的目录；第四个参数：相当于过滤器filter
      • QFileDialog() 文件对话框的对象exec_()方法来选择文件，并将文件的内容显示在文本编辑控件中；
    • Pixmap()函数：加载并呈现本地图像

9. 窗口绘图类控件

  1. PyQt5实现绘图功能？

    • 通过QPainter、QPen、QBrush三个类实现绘图功能；
    • QPixmap：作用是加载并呈现本地图像，而图像呈现的本质也是通过绘图方式实现的；

  2. QPainter：绘制从简单的直线到复杂的饼图；

    1. 绘制操作：在QWidget.PaintEvent()中完成；

    2. 绘制方法：必须放在QtGui.QPainter对象的begin()和end()之间；

    3. 绘制方法和画笔格式：书中有表格展示；

    4. 通过绘制点实现正弦函数

  3. QPen: 是一个基本的图形对象，用于绘制直线、曲线、轮廓画出矩形、椭圆形、多边形和其他形状。

    • pen = QPen(Qt.black, 2, Qt.SolidLine) # 穿件了QPen对象。黑色、2像素、SolidLine预定义线条样式；
    • CustomDashLine自定义的线条样式；
      • setDashPattern()方法使用数字列表定义样式：列表中个数必须是偶数

  4. QBrush：绘制九个不同样式的矩形

    • 将Qprinter对象的画刷设置成QBrush对象
    • 调用drawRect()方法绘制矩形

  5. QPixmap

10. 拖拽和剪切板

  1. Drag与Drop

    • 拖拽功能是基于MIME类型的拖拽数据传输的，其类型是基于QDrag类；QMimeData对象将关联的数据与其对应的MIME类型相关联；
    • MimeData类有相关的函数处理MIME类型的数据：判断函数、设置函数、获取函数；
    • QWidget对象支持拖拽动作：允许拖拽数据的控件必须设置QWidget.setDragEnabled()函数为True;
    • 实例：将左边文本框的输入拖拽到右边的下拉菜单栏中——实现下拉菜单的动态输入；
  2. QClipboard
    • QClipboard 例子: 对系统剪切板的访问，可以在应用程序之间复制和粘贴数据；
    • 文本数据(Text)、图像数据(Image)、网页数据（HTML）;
    • 当剪切板内容发生变化的时候，dataChange信号被发射；

11. 日历与时间

  1. QCalendar：日历控件，提供了一个基于月份的视图，允许用户通过鼠标和键盘选择日期；

    • QCalendar的对象是通过QCalendarWidget类进行创建的；
    • 通过QDate()方法设置具体的日期；
    • 从窗口中选定一个日期，会发射一个QtCore.QDate信号，将此信号连接到用户的槽函数中；

  2. QDateTimeEdit：允许用户编辑日期和时间的控件；

    • setDisplayFormat()：设置时间的格式；

    • time()：返回编辑的时间；data()：返回编辑的日期；

    • 初始化QDateTimeEdit不指定日期时间时

      • 系统分配一个默认时间2000年1月1日 0时0分0秒；

      • currentDateTime()，设置日期时间为本机此刻的日期和时间；

    • setCalendarPopup：设置日期的选择为弹出框内进行选择；

    • 两个函数模板：

      • 信号与槽函数的连接的函数模板：同样分为日期、日期和时间、时间，三种槽函数；
      • 获取日期、时间的函数模板；

12. 菜单栏和工具栏

  1. 菜单栏QMenu：分清上一级是谁；

    • menuBar()：创建一个menuBar菜单栏对象；

    • addMenu(“File”)：在菜单栏中添加一个新的QMenu对象；

    • QAction()：创建一个动作；

    • addSeparator() : 在菜单栏中添加横线，在需要添加横线的动作后面添加；

    • 添加QAction两种方式实现：可以使字符串或者是QAction对象；

    • 基于file的第二个动作：为动作Action添加一个快捷键的方法；

  2. 工具栏QToolBar：由文本、图标或者其他空间按钮组成的可移动面板

    • addToolBar()方法：在工具栏区域添加文件工具栏；
    • 将信号连接到槽函数；每当点击工具栏中的按钮时，都会发射actionTriggered信号。这个信号将关联的QAction对象 的引用发送到连接的槽函数;
    • QToolBar类中常用的方法（）

  3. 状态栏QStatusBar

    • 实例：创建一个file菜单栏，并为其添加一个动作，并为其创建一个编辑框；最后点击show动作时，提示已被点击；

    • 保留一个水平条，作为状态栏，用于显示永久的或临时的状态信息；

    • 通过主窗口的QMainWindow的setStatusBar()函数设置状态栏
