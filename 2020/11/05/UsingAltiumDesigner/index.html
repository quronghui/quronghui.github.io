PCB 中层的概念

  1. PCB中各层的介绍
    • Internal plane layer 内部电源/接地层，仅用于多层板；
    • midlayers：中间信号层，用于多层板设计
  2. 一到八层电路板的叠层设计方式
    • 六层板的叠层，推荐叠层方式：SIG－GND－SIG－PWR－GND－SIG；
  3. 差分信号
    • 信号的振幅相同，相位相反
    • 差分走线必须是等长、等宽、紧密靠近、且在同一层面的两根线；差分线对之间要有 GND 隔离，或者保持距离，不要太近。
    • 一般类型：USB、以太网、PCIE、SATA、RS485、RS422、HDMI、LVDS
  4. Layer Stackup
  5. mark点：电路设计中PCB应用于自动贴片机上的位置识别点
    • mark点一般都是放置于BGA封装器件的对脚，将焊盘改为的mark点。

六层板实操体会

  1. 多层板和双层板的误区？
    • 双层板：Top和Bottom，双层布线，双面铺铜；所以会觉得双层板是不是会更加简单；
    • 多层板：除了Top和Bottom，添加了信号层、地层、电层；所以会觉得多层板会很难；
    • 其实不然，多层板多了偶数信号层，这样在进行布线的时候我们就可以在不同的层进行穿梭，相当于减少了布线的难度，只是生产的时候价格相对会高一些；
  2. 多增加了信号层怎么使用呢？
    • 类似于Top和Bottom进行布线就可以，没有其他的要求；
  3. 电层和地层怎么使用呢？
    • 电层：相当于一块铜皮，通过布置一个密闭型框后，选择对应的电源层后，通过打孔连接地层和top/bottom层的电源，便于电源的连接，而且增加了电源的面积；
    • 地层：同样是一块铜皮，通过布置一个密闭型框后，选择不同的GND后，通过打孔连接地层和top/bottom层的地，便于不同地的连接。
  4. Top和Bottom层使用一个屏蔽的Earth?
    • 增加一块屏蔽层，便于信号间的隔离。
  5. PCB布线的技巧？
    • 优先进行差分信号的走线，保证其等距离，同一层；
    • 优先进行主芯片（最多信号线的模块），
    • 布线的时候尽量横平竖直，模块化布线，同一个区域尽量走同一种信号线；
