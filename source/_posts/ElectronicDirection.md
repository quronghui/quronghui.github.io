### 嵌入式学习方向

1. CPU 方向：
   + 计算机 / SOC 架构；
   + 脚本语言，Python 优先；
2. 物联网方案
   + 悉 TCP 与 UDP 的特点与使用，了解 Wi-Fi / 蓝牙协议优先；
   + FreeRTOS / uCOS 等嵌入式系统优先；
   + 熟悉脚本语言，掌握 Python 优先；
   + 熟悉计算机网络协议熟悉 802.11 协议或蓝牙协议优先；
   + 熟悉 GCC 和 GDB；
3. 熟悉系统外围驱动（I2C、SPI、UART、485）和应用的编程
